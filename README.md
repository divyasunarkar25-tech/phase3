# Random Quote Generator

A simple and interactive web application that generates random inspirational quotes with a single click. Built using HTML, CSS, and JavaScript.

## 🚀 Features

- **Random Quotes:** Displays a new quote every time you click the button.
- **Responsive & Simple UI:** Clean and straightforward user interface.
- **Beginner Friendly:** Great example of basic DOM manipulation in JavaScript.

## 🛠️ Technologies Used

- **HTML5:** For the structure component of the web page.
- **CSS3:** For basic styling and layout (`style.css`).
- **JavaScript:** For the logic to select and display random quotes (`main.js`).

## 📁 Project Structure

```text
quotes/
├── index.html   # Main HTML file containing the app structure
├── style.css    # CSS file for styling the UI
├── main.js      # JavaScript file containing quote generation logic
└── README.md    # Project documentation
```

## 💻 How to Use

1. Go to the `quotes` folder on your machine.
2. Open the `index.html` file in any modern web browser (Chrome, Firefox, Edge, etc.).
3. Click the **"New Quote"** button to see a random inspirational quote!
